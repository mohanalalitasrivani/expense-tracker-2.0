# Expense Tracker 2.0 using Python

## Description
Expense Tracker 2.0 is a simple command-line application developed using Python.  
This project helps users manage expenses by adding, viewing, searching, and calculating expenses based on categories and months.

---

## Features
- Add expenses
- View all expenses
- Search expenses by category
- Calculate total expenses by category
- Calculate monthly expenses
- Menu-driven CLI application

---

## Technologies Used
- Python 3

---

## File Name
expense_tracker.py

---

## How to Run
1. Open the Python file
2. Run the program
3. Select options from the menu

Example:
1 → Add Expense  
2 → View Expenses  
3 → Search by Category  
4 → View Category Total  
5 → View Monthly Total  
6 → Exit
